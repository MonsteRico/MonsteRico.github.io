How to Make a Level

The whole basis of this game is that the level editor is Paint. You can use any photo editing program, but if you are on windows I recommend Microsoft Paint which is likely on your system already. If not, any other photo editing software will work. Start by resizing your canvas to 64x32 pixels. Zoom in so that you can see easily. Choose a pencil tool and start painting your level. The colors to use can be found below, with each color doing something different. Make sure that you use the exact color codes or RGB values of the color or the level will not work right. If you are using Paint, the colors below are the exact same and in the same spots. So the color on the top row at the very end in Paint is the player start color. Make sure each level has a Goal, the Player, and a way for the Player to get to that goal. When you are finished you can upload your level. Make sure to not use any personal information in the levels name or author. For author, you should use your username in a game or something similar. Have fun and if you have any questions or encounter a bug then use the Report a Bug page to ask or tell away. If you happen to know me through something feel free to DM or just ask me as well!
